setWebhook

curl -XPOST -d'{"url": "http://example.com/webhooks", "webhook_key": "zbb5y4PZ98R8fW4w"}' \
  http://messaging.livetex.ru/<version>/setWebhook?key=<authentication_key>

Настройка оповещений. Оповещения будут присылаться по указанному при конфигурации URL. В теле запроса будет присутствовать webHookKey, который позволит идентифицировать источник оповещения на принимающей стороне. В ответ на запрос сервер ожидает 200 HTTP ответ. В противном случае сервер оповещений постарается повторить оповещение, сервер пытается повторно послать оповещение в течение 24 часов.

Формат запроса

Имя Тип Обязательный Комментарий
url String Да URL по которому будут отсылаться WebHooks оповещения, поддерживаются только HTTPS соединения.
webhook_key String Да Ключ аутентификации передаваемый во всех сообщениях оповещения.

Формат ответа

Имя Тип Обязательный Комментарий
success Boolean Да Успешно ли выполняется запрос.
error_message String Нет Описание ошибки.
error_code Integer Нет Код ошибки. Для возможности интернационализации.
