sendText

Пример

curl -XPOST -d'{"user_id": "<user_id>", "text": "Hello World!"}' \
  http://messaging.livetex.ru/<version>/sendText?key=<authentication_key>

Отправка текста в обращение от лица собеседника. В случае успешного запроса создается оповещение UserText, доступное собеседнику и операторам, обслуживающим обращение.

Формат запроса

Имя Тип Обязательный Комментарий
user_id String Да Идентификатор собеседника в точке контакта. Задается на стороне клиента
text String Да Текстовое сообщение, отправленное собеседником

Формат ответа

Имя Тип Обязательный Комментарий
success Boolean Да Проверка успешности выполнения запроса
message_id String Да Идентификатор, присвоенный сообщению
timestamp Long Да Время в формате Unix time
error_message String Нет Описание ошибки
error_code Integer Нет Код ошибки. Для возможности интернационализации
