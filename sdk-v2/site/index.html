LiveTex Mobile SDK 2.x

LiveTex Mobile SDK 2.0 предоставляет набор инструментов для организации консультирования пользователей мобильных приложений.

Функциональность

  • Возможность выбора департамента и/или оператора для обращения.
  • Передача и прием текстовых сообщений.
  • Передача и прием файлов.
  • Мгновенное оповещение о событиях: оповещение о наборе, удерживающие сообщения, изменение оператора диалога, закрытие диалога оператором.
  • Просмотр истории сообщений.
  • PUSH-уведомления.

Концепции

  • Собеседник - пользователь мобильного приложения, которому предоставляется консультация.
  • Оператор - сотрудник организации клиента, который осуществляет консультацию.
  • Направление обращения - уточняющие атрибуты обращения, необходимые для корректной маршрутизации на оператора.
  • Собеседник может иметь только одно активное обращение.
  • Собеседник может начать переписку в независимости от доступности Операторов на рабочем месте.
  • Сообщения собеседника будут доставлены, когда необходимый оператор станет доступен для обработки обращений.

Архитектура

Клиентские библиотеки LiveTex SDK состоят из двух компонент:

  • LiveTex Core - базовая функциональность, необходимая для интеграции с сервисами LiveTex.
  • LiveTex Kit - библиотека готовых UI-элементов.

Быстрый старт

  1. Создать аккаунт в системе LiveTex.
  2. Создать операторов.
  3. Создать точку контакта для мобильного приложения.
  4. Запросить ключ разработчика у техподдержки по email. В письме необходимо указать идентификатор аккаунта в системе Livetex. Идентификатор аккаунта в системе можно посмотреть в личном кабинете в выпадающем списке при нажатии на профиль администратора (верхний правый угол).
  5. Скачать демо приложение
